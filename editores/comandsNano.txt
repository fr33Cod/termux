# NANO COMMANDS

💾 Save & Exit
Ctrl + O      → Save file
Ctrl + X      → Exit
Ctrl + C      → Show cursor position
Ctrl + X + N  → Exit without saving

📋 Navigation
Ctrl + A      → Beginning of line
Ctrl + E      → End of line
Ctrl + Y      → Page up
Ctrl + V      → Page down
Ctrl + _      → Go to line number

✂️ Editing
Ctrl + K      → Cut line/block
Ctrl + U      → Paste
Ctrl + 6      → Set mark (start selection)
Alt + 6       → Copy line/block
Alt + U       → Undo last action
Alt + E       → Redo undone action

🔍 Search / Replace
Ctrl + W      → Search text
Ctrl + \      → Replace text
Alt + W       → Repeat last search

🧹 Cleaning tricks
Ctrl + K (repeated)

Delete entire file:
Ctrl + Y
Ctrl + 6
Ctrl + V
Ctrl + K
